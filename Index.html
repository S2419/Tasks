Api name Description Input
Earthquakes This API shows a list of earthquakes and ranks them in size of magnitude. Parameters for this: north, south, east, west: coordinates of bounding box.
Streetname Lookup Can search up the street name anywhere in the world by querying it. Parameters are q (query term, URL encoded) which is any location or word beginning.
Country info Gives you in-depth information about a country from currency to what continent it's on. The parameters for this are just the country/country code.
