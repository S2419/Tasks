Api name Description
Earthquakes This API shows a list of earthquakes and ranks them in size of magnitude. Parameters for this: north, south, east, west: coordinates of bounding box.
Streetname Lookup Can search up the streetname anywhere in the world by querying it. Parameters are q (query term, url encoded) which is any location or word beginning.
Country info Gives you indepth informtion about a country from currency to what continent its on. The parameters for this are just the country/country code.
